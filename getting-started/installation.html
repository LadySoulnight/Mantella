Is my OS supported?
We are actively supporting all big operation systems (Unix and Linux variants, Mac OS X and Windows), provided all necessary dependencies are already installed.
{% include h2 title="Prerequirements" %}
  • GCC 4.8.1, Clang 3.3 or any other C++11 feature complete compiler
  • Armadillo C++ minimum required is 6.400.0

Installing Armadillo C++ on Ubuntu (visit Armadillo's download page to find installation instructions for other operating systems):

{% highlight bash %} sudo apt-get install libblas-dev liblapack-dev libopenblas-dev wget --quiet -O armadillo.tar.gz http://downloads.sourceforge.net/project/arma/armadillo-6.200.4.tar.gz mkdir armadillo tar -xzf armadillo.tar.gz -C ./armadillo --strip-components=1 cd armadillo cmake . make sudo make install {% endhighlight %} {% include h2 title="Download and Install" %}

Mantella is a header-only library, downloading the latest stable version and placing the header files into a directory standardly searched by your compiler finishes the installation.

{% highlight bash %} git clone --depth 1 --branch master https://github.com/SebastianNiemann/Mantella.git cd Mantella cmake . make -j 4 sudo make install {% endhighlight %} Done. You can now start creating applications with Mantella.
