--- layout: page --- {% include h1 title="Random Matrix Generator" %} {% include api call="getRandomRotationMatrix" returns="arma::Mat<T>" params="unsigned int N" %}

Generates a randomly and uniformly drawn N-dimensional rotation matrix.

  • Note: The rotation of each dimension will be uniformly distributed.
  • Requirement: The dimension N must be greater than or equal to 2.
{% include api call="getRandomPermutation" returns="arma::Col<unsigned int>" params="unsigned int N, unsigned int K" %}

Generates a randomly and uniformly drawn permutation of length K from N elements.

  • All elements in the permutation will be unique and within \([0, N - 1]\).
  • Requirement: The length K must be greater than or equal to 1.
  • Requirement: The length K must be less than or equal to the number of elements N.
{% include api call="getRandomPermutation" returns="arma::Col<unsigned int>" params="unsigned int N" %}

Generates a randomly and uniformly drawn permutation of length N from N elements.

  • All elements in the permutation will be unique and within \([0, N - 1]\).
  • Note: The result is equal to getRandomPermutation(N, N).
  • Requirement: The number of elements N must be greater than or equal to 1.
