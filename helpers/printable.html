Classes that extend mant::Printable must implement a member function .toString(), returning a filesystem friendly name for the class.

{% highlight cpp %} #include #include int main() { unsigned int numberOfDimensions = 10; std::shared_ptr optimisationProblem = std::shared_ptr(new mant::bbob::LinearSlope(numberOfDimensions)); mant::HillClimbing hillClimbing(optimisationProblem); arma::Mat results; /* Doing something, filling the results matrix */ // Save the results results.save("/some/path/" + mant::to_string(optimisationProblem) + "_dim" + std::string(numberOfDimensions) + ".mat", arma::raw_ascii); return 0; } {% endhighlight %} Choosing arma::raw_ascii as file output format guarantees compatibility with Matlab (using dlmread('/path/to/file');). If this is not necessary for you, we recommend using arma::arma_binary, which operates faster and generates shorter files. {% include api call=".toString" returns="std::string"%}

Returns a filesystem friendly string representation of the derived class.

{% include api call="to_string" returns="std::string" params="std::shared_ptr<mant::Printable> P" %}

Syntactic sugar, added to have a function call similar to C++ STLs std::to_string(...).

  • This is equal to P->toString().
